# vue-gen-form-1

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Install copy-files-from-to to copy package.json to dist folder

```sh
npm install -g copy-files-from-to
```

### Run copy-files-from-to after build

```sh
copy-files-from-to
```

### Publish

```sh
cd dist
npm publish
```
